# MoYu Novel Reader (摸魚看小說) 🐟

一款專為桌面工作環境設計的「隱身」小說閱讀器。具備高度自定義的透明度控制、自動章節解析、讀書記錄保存及極致的隱身模式。

## ✨ 主要功能 (Key Features)

- **📖 智能載入**: 支持載入 TXT 文本，並根據常見的正則表達式自動生成小說目錄。
- **📚 書庫管理**: 自動保存已加載的小說及閱讀進度（JSON 持久化），下次打開自動跳回上次觀看位置。
- **👁️ 極致隱身 (Stealth Mode)**:
  - **100% 透明背景**: 進入閱讀模式後背景完全消失。
  - **鼠標追蹤隱藏**: 鼠標離開文本區域時文字自動消失，移回時瞬間恢復。
  - **防點擊穿透**: 即使背景全透明，依然能精準捕捉滾輪滾動，不會誤觸後方的窗口。
- **⚙️ 高度自定義**:
  - 實時調整字體大小、顏色、文字透明度。
  - 控制窗口背景不透明度。
  - **所見即所得**: 設置介面自帶預覽框，調整樣式立即顯示效果。
- **🌐 雙語支持**: 提供中英雙語界面切換。
- **🚀 快捷開啟**: 通過 `StartReader.bat` 一鍵靜默啟動到系統托盤。

## 🛠️ 安裝與運行 (Installation)

1. **環境要求**:
   - Python 3.9+ 
   - 安裝依賴: `pip install PyQt6 chardet`

2. **啟動方式**:
   - **雙擊 `StartReader.bat`**: 程序將在背景啟動，圖標位於右下角系統托盤。
   - **系統托盤操作**: 右鍵點擊托盤圖標可選擇打開書庫、閱讀器或退出。

## 🖥️ 使用介紹

1. **添加小說**: 在書庫界面點擊「添加小說 (TXT)」。
2. **選擇章節**: 選中小說後，右側會出現章節下拉選單，自動定位到你上次離開的章節。
3. **進入閱讀模式**:
   - 點擊「開始閱讀」打開閱讀器。
   - 點擊閱讀器頂部的「模式 ✨」按鈕進入隱身狀態。
   - 鼠標移開即隱身，移回即恢復。
