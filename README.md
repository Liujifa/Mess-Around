# MoYu Novel Reader (摸鱼看小说) 🐟

一款专为桌面工作环境设计的“隐身”小说阅读器。具备高度自定义的透明度控制、自动章节解析、智能网页小说下载（带防爬虫绕过功能）、读书记录保存及极致的隐身黑客模式。

## 🛠️ 使用技术 (Technologies)

本项目使用以下技术：
- **Python 3.9+**: 核心编程语言，提供文本处理和跨平台运行能力。
- **PyQt6**: 构建桌面图形用户界面（GUI）。
- **Requests & BeautifulSoup4**: 处理网络请求与 HTML 解析，实现网页小说抓取。
- **browser-cookie3**: 用于获取浏览器 Cookie 以抓取防爬虫网站。
- **Regex (正则表达式)**: 提取和匹配小说网站的章节格式。

## ✨ 核心功能 (Features)

- **🌐 网页小说下载**:
  - **自动解析目录**: 输入单章或目录链接，下载器会尝试解析并下载全本小说。
  - **动态网页支持**: 内置 AJAX 解析逻辑，可获取部分采用动态加载的网站正文（如 `xpxs.net`）。
  - **编码自动识别**: 支持并转换 GBK / UTF-8 等网页编码。
- **📖 本地载入**: 支持导入本地 TXT 文本，并生成源生章节目录。
- **📚 书库进度保存**: 采用 JSON 格式记录本地进度。关闭窗口或跳转章节时自动保存。
- **👁️ 隐身模式 (Stealth Mode)**:
  - **背景透明**: 开启隐身模式后，窗口背景完全透明。
  - **鼠标追踪隐藏**: 鼠标离开文本区域时，文字自动隐藏；移回时恢复。
  - **系统级隐藏**: 窗口不在任务栏显示。
- **⚙️ 界面自定义**:
  - 调整字体大小、字体颜色、文字透明度，并带有实时预览框。
- **🌐 界面双语切换**: 支持切换中文/English。

## 🖥️ 使用方法 (Usage)

1. **一键启动**: 双击运行目录下的 `StartReader.bat`，程序将静默启动并在右下角系统托盘生成一条“小鱼”图标。
2. **打开书库**: 右键点击托盘图标，选择“小说书库 / 网页下载”。
3. **添加小说**:
   - 点击“添加小说(TXT)”导入本地文件。
   - 或点击“网页下载 TXT”粘贴小说目录连接，全自动抓取并入库。
4. **开始阅读与隐身**:
   - 在书库中双击小说进入阅读器。
   - 拖拽阅读器任意位置移动窗口。点击顶部的“阅读模式”即可进入隐身形态。
   - 鼠标滚轮丝滑翻页，利用右键或顶部菜单进行快捷章节跳转。
