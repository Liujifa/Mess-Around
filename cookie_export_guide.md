# How to Export Browser Cookies 🍪

Since modern browsers (Chrome 127+, Edge) have introduced **App-Bound Encryption**, MoYu cannot read your cookies directly anymore. You need to manually export them using a browser extension.

## Step 1: Install a Cookie Extension
We recommend **Cookie-Editor**, which is available for all major browsers:
- [Chrome Web Store](https://chrome.google.com/webstore/detail/cookie-editor/hlkenndednpecjcgjomegrklhggfmlgn)
- [Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/cookieeditor/neaplpkndocofebhcennajmhljjnechk)
- [Firefox Browser Add-ons](https://addons.mozilla.org/en-US/firefox/addon/cookie-editor/)

## Step 2: Export Cookies
1.  Open your browser and navigate to the novel website (e.g., `poshuoge.com`).
2.  Log in or pass any "Verify you are human" checks.
3.  Click the **Cookie-Editor icon** (usually a small cookie 🍪) in your browser toolbar.
4.  Click the **Export** button at the bottom of the extension popup.
5.  Select **JSON** or **Netscape** as the format.
6.  The cookie data is now copied to your clipboard. **Paste it into a new text file** (e.g., `cookies.txt`) and save it on your computer.

## Step 3: Import into MoYu
1.  In the MoYu download dialog, select **"Import from file (.txt / .json)"** from the browser dropdown.
2.  Click **Download**.
3.  Select the `cookies.txt` file you just saved.
4.  MoYu will read the cookies and proceed with the download!

# 如何导出浏览器 Cookie 🍪

现代浏览器（如 Chrome 127+ 和 Edge）引入了 **App-Bound Encryption** 安全机制，导致 MoYu 无法直接读取您的 Cookie。您需要使用浏览器扩展手动导出它们。

## 第一步：安装 Cookie 扩展
推荐使用 **Cookie-Editor**，支持各大主流浏览器：
- [Chrome 网上应用店](https://chrome.google.com/webstore/detail/cookie-editor/hlkenndednpecjcgjomegrklhggfmlgn)
- [Microsoft Edge 加载项](https://microsoftedge.microsoft.com/addons/detail/cookieeditor/neaplpkndocofebhcennajmhljjnechk)
- [Firefox 浏览器扩展](https://addons.mozilla.org/en-US/firefox/addon/cookie-editor/)

## 第二步：导出 Cookie
1.  在浏览器中打开目标小说网站（例如 `poshuoge.com`）。
2.  登录账户或通过“我是人类”验证。
3.  点击浏览器工具栏中的 **Cookie-Editor 图标**（通常是一个小饼干 🍪）。
4.  点击弹出窗口底部的 **Export (导出)** 按钮。
5.  选择 **JSON** 或 **Netscape** 格式。
6.  此时 Cookie 已复制到剪贴板。**将其粘贴到新建的文本文件中**（例如 `cookies.txt`）并保存到电脑。

## 第三步：导入 MoYu
1.  在 MoYu 下载对话框中，从浏览器下拉列表中选择 **“从文件导入 (.txt / .json)”**。
2.  点击 **下载**。
3.  选择您刚才保存的 `cookies.txt` 文件。
4.  MoYu 将读取 Cookie 并开始下载！
